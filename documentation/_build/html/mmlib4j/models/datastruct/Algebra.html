Algebra¶

public class Algebra¶

This class is a wrapper to some netlib BLAS methods.

Author:Charles Ferreira Gobber

See also: AbstractMatrix, Float32, Float64, BLAS

Methods¶

dminus¶

public static Matrix<double[]> dminus(int size, Matrix<double[]> A, Matrix<double[]> B)¶

This method perform summation between two matrix for double type. Let \(A\) and \(B\) two matrix. Thus, this operation performs:

\[A = A - B.\]

Observation: This operation overwrites the matrix \(A\).

Parameters:
  • size – Size matrix (rows x columns).
  • A – A matrix of type double[].
  • B – A matrix of type double[].
Returns:

A matrix \(A = A - B\).

multAdd¶

public static Matrix<float[]> multAdd(float alpha, Matrix<float[]> A, Matrix<float[]> B, Matrix<float[]> C)¶

This method perform multiplication between two matrix for float type. Let \(A\) and \(B\) two matrix. Thus, this operation performs:

\[C=\alpha A * B + C\]

where \(\alpha\) is a scalar and C the result matrix. Note that, if \(C\) is a zero matrix and we set \(\alpha=1\) then the we have \(C=A * B\).

Parameters:
  • alpha – A float scalar.
  • A – A matrix of type float[].
  • B – A matrix of type float[].
  • C – A matrix of type float[].
Returns:

A matrix \(C=\alpha A * B + C\).
